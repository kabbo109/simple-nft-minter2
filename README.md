# Simple NFT Minter

A flat-structure, professional-grade NFT project. This repository implements an ERC-721 Token with URI storage capabilities, allowing for the minting of unique digital assets. It includes utilities for generating metadata and deploying to Ethereum testnets.

## Features

- **ERC-721 Standard:** Fully compliant Non-Fungible Token contract.
- **URI Storage:** Individual metadata URIs for each token.
- **Ownership Controls:** Restricted minting access using Ownable pattern.
- **Metadata Generator:** Utility script to create standard JSON attributes.
- **Gas Optimized:** Uses OpenZeppelin's optimized base contracts.
